## Getting started

**1. install bootstrap, fontawesome and jquery**
```
bower install
```

**2. Install the Node dependencies as root/Administrator.**
```
sudo npm install
```

**3. copy fonts from source to public and copy bootstrap variables for custom.**
```
grunt init
```

**5. let the magic begin.**
```
gulp
```
